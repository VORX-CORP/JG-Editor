About JG-Editor

ABOUT THIS PROJECT

Authored By: JOE GANES
Project Started: March 2026

This application was born out of a need for high-performance dev tools in restricted environments. When security policies make simple tasks—like installing a "Pretty Print" plugin—a multi-day IT process, productivity suffers.

This is a zero-footprint, standalone HTML solution designed for speed and privacy. It provides robust XML and EDI validation and formatting without the need for installation or administrative privileges. Your data never leaves your machine, and the entire suite can be run directly from a web browser or a portable USB drive.
